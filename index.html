Syntaxblocks

Experimental UI for mobile touchscreen code viewing/authoring

Download this project as a .zip file Download this project as a tar.gz file

Specify program behavior from your phone, without input frustrations

Coding on the screen of your mobile device can be painful. This is a silly little thing I thought of on the train to see if I could get anywhere with this problem.

ASTs and SPTs

It should be possible to specify program behavior with just swipes and large-target taps. I think. Skipping the code and working directly with the Syntax Tree seems like it could be a good start. Interpreting the Syntax Tree as a Space Partitioning Tree seems like it'd be an efficient way to display the information on a tiny screen.
